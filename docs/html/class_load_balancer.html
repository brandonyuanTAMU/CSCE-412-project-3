Detailed Description

Manages a pool of web servers and a queue of incoming requests.

The LoadBalancer distributes requests to available servers, dynamically scales the server pool based on queue thresholds (50*servers to 80*servers), blocks requests from specified IP ranges, tracks statistics, and logs all events to both the terminal (with color) and a log file.

◆ tick()

void LoadBalancer::tick ( )

Advances the simulation by one clock cycle.

Advances the load balancer by one clock cycle.

Processes active servers, checks scaling thresholds, and assigns queued requests to available servers.

Each tick performs three actions in order:

  1. Ticks all busy servers and logs completed requests.
  2. Checks if the scaling cooldown has elapsed, then adds or removes servers based on queue-to-server ratio thresholds.
  3. Assigns queued requests to any newly available servers.
